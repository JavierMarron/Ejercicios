Preguntas y Respuestas

¿Cuál es el lenguaje de marcado utilizado para crear esta página?

El lenguaje de marcado utilizado es HTML (Hypertext Markup Language).

¿Qué significa CSS?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo que se utiliza para describir cómo se presenta un documento HTML en la pantalla, dispositivo o medio de impresión.

¿Puedo usar varios selectores a la vez para aplicar un mismo estilo?

Sí, puedes utilizar varios selectores a la vez separados por comas para aplicar el mismo estilo a múltiples elementos HTML. Por ejemplo: `.pregunta, .respuesta { font-size:16px; }`
